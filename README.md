### fonts directory
- Windows: `C:/Windows/Fonts/` or `C:/Users/〈user_name〉/AppData/Local/Microsoft/Windows/Fonts/`
- macOS: `/Library/Fonts` or `/Users/〈user_name〉/Library/Fonts`
